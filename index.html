Intro and description

Philosophies

CSS3

I have no real interest in using backwards and broken technologies. My focus is to try to make this lib as simple as possible using current technologies, so if you need <= IE9 you may need to look elsewhere.

Icons

I've included Fontawesome because it is a great library with tons of beautiful icons. 'Nuff said.

Accessibility

I'm learning about and experimenting with Accessibility. I'll try to improve where I can, as much as I can.

Support

At this point, I'm much more interested in learning and creating than maintaining this as a project proper. That said, feel free to fork, add and remove, and submit a PR to see if that's something that others might also benefit from.

Alerts

.alerts

They are transparent by default.
They should be used to provide information. Green may or may not be equated with success messages.

Blue could be informational messages that are not critical. Maybe add a link for more information.

Yellow is usually used for issues that do not require immediate action.
Red can be the most 'serious' of alerts.
Neutral info. Like something about how about cookies are used for your website...
